# Raw data

Large raw data files are not stored in this repository because they exceed GitHub upload limits.

Download manually from:

https://huggingface.co/datasets/petchthwr/ATFMTraj

Required ATFMTraj files:

- ESSA_stats.csv
- ESSA_TEST_X.tsv
- ESSA_TEST_Y.tsv
- ESSA_TEST_Z.tsv

Place them in this folder when working locally or inside GitHub Codespaces.
