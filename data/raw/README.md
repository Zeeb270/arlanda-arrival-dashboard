# Raw data

This folder is reserved for raw aviation datasets.

Large raw files are not committed to GitHub because they exceed normal repository file-size limits.

Required ATFMTraj files:

- ESSA_stats.csv
- ESSA_TEST_X.tsv
- ESSA_TEST_Y.tsv
- ESSA_TEST_Z.tsv

Download source:

https://huggingface.co/datasets/petchthwr/ATFMTraj

For the first setup, only ESSA_stats.csv should be uploaded to GitHub.
The large TSV files should be downloaded inside GitHub Codespaces or stored locally.
